6  Codebook

Below is the survey codebooks

Below you will find relevant information about all variables in the datafiles:

  • source = Where did the column come from?
  • var_category = What category of data is it?
    • ID = identifiers
    • Survey Data = information about survey engagement and completion
    • Condition = manipulated conditions
    • Demographics = unique data linked to participant
    • Demographics (Calculated) = unique data linked to participant; calculated or aggregate
    • Course Experience = Quantitative data on overall course experiences
    • Personal Beliefs = Quantitative data on general experiences
    • Practices = The practices collected
  • var_type = What type of data is it?
  • var_original_name = The original variable name in cleaning files
  • var_original_name = The updated variable name in cleaning files
  • var_intro_wording (optional) = Wording provided to participant prior to completing the item
  • var_wording = Item wording provided to participant OR description of calculated variable
  • var_scale = Response scale information
  • var_notes (optional) = Any other relevant information about the variable
  • var_reference (unused at this time) = TBD

You can search using the search button in the top right. Example search terms:

  • “logical” = find all binary T/F variables
  • “Demographics” = find all demographic items collected
  • “A14” = find all variables that start with A14
